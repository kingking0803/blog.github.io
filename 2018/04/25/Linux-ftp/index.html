LINUX ftp

VSFTPD安装

首先查看是否安装了vsftpd

rpm -q vsftpd

如果没有安装则执行以下命令，进行安装

yum install -y vsftpd

查看vsftpd服务状态

service vsftpd status

配置vsftpd.conf

首先备份配置文件

cp /etc/vsftpd/vsftpd.conf /etc/vsftpd/vsftpd.conf.bak

编辑该配置文件

vim /etc/vsftpd/vsftpd.conf

修改并添加如下内容

anonymous_enable=NO # 禁用匿名登录
ascii_upload_enable=YES
ascii_download_enable=YES
chroot_local_user=YES # 启用限定用户在其主目录下

#之后添加下列内容到vsftpd.conf末尾
use_localtime=YES
listen_port=21
chroot_local_user=YES
idle_session_timeout=300
guest_enable=YES
guest_username=vsftpd
user_config_dir=/etc/vsftpd/vconf
data_connection_timeout=1
virtual_use_local_privs=YES
pasv_min_port=10060
pasv_max_port=10090
accept_timeout=5
connect_timeout=1

建立用户

第一行用户名，第二行密码，不能使用root为用户名

vim /etc/vsftpd/virtusers

添加内容

test
123456
su
123456

生成数据文件

首先你需要确定你是否有db_load这个命令

rpm -q db

安装之前先查询一下

yum search db4

安装

yum install -y compat-db47.x86_64 (以你的机器为准)

生成db文件（每次添加新的用户都要重新生成一遍这个文件，因为这个文件相当于是用来校验用户账号密码是否正确）

db_load -T -t hash -f /etc/vsftpd/virtusers /etc/vsftpd/virtusers.db

设定PAM验证文件，并指定对虚拟用户数据库文件进行读取

chmod 600 /etc/vsftpd/virtuser.db

修改/etc/pam.d/vsftpd文件

修改前先备份

cp /etc/pam.d/vsftpd /etc/pam.d/vsftpd.bak

将里面所有东西都注释，并添加如下内容（如果系统为32位，上面lib64改为lib）

auth sufficient /lib64/security/pam_userdb.so db=/etc/vsftpd/virtusers
account sufficient /lib64/security/pam_userdb.so db=/etc/vsftpd/virtusers
